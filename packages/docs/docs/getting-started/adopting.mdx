# Adopting in an existing application

Integrating **superstate** in an existing application is easy and harmless. In fact, all you have to do is to
[install superstate](/getting-started/installation) and wrap your next state within `superstate()`.
[Learn how to create your first superstate &rarr;](/getting-started/first-state)

:::info Tips and tricks
You can use [Broadcasting](/advanced/broadcasting), [Extensions](/getting-started/extensions) and/or [Middlewares](/advanced/middlewares)
to update your legacy states when a **superstate** changes.
:::
