# Installation

All you have to do is installing `@superstate/core` as any other package:

<Tabs>
<TabItem value="bash" label="npm">

```bash
npm install @superstate/core
```

</TabItem>
<TabItem value="yarn" label="yarn">

```bash
yarn add @superstate/core
```

</TabItem>
</Tabs>
